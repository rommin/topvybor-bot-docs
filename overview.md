## Анализ текущего состояния бота

### Что уже есть
- Список 140+ МФО с инструкциями по отписке
- Парсинг SMS/выписки (базовый)
- Раздел "Надёжные МФО" (список без офферов)
- Раздел "Чарджбек"
- Дейли-рассылка (пустой пинг)
- Логирование событий в PostgreSQL
- Статистика в Telegram-группу

### Главные проблемы

| Проблема | Следствие |
|----------|-----------|
| Разовый сценарий — отписался и ушёл | Высокий % блокировок |
| Дейли-рассылка без ценности | Раздражение, блокировки |
| Нет follow-up после инструкции | Нет обратной связи, нет удержания |
| SMS-парсинг не покрывает все форматы | Пользователь не получает помощь |
| Нет монетизации (офферы без UTM, без квиза) | Нет дохода |
| Нет генератора писем/заявлений | Пользователь уходит делать сам |
| Нет жалоб в ЦБ / Роспотребнадзор | Конкуренты предлагают |

## Монетизация — рекламные сценарии МФО

### Принцип: контекстная реклама, не баннеры
Реклама работает только в нужный момент — когда пользователь к этому готов.

### Сценарий R1: После успешной отписки (самый эффективный)
```
Отличная работа! Вы отписались от [Сервис].
Сэкономили ~590 руб/мес = 7 080 руб/год 🎉

Если когда-нибудь понадобится займ — вот МФО без скрытых подписок:
• Займер — до 30 000 руб, без подписок ➜ [ссылка UTM]
• Екапуста — первый займ бесплатно ➜ [ссылка UTM]

Это партнёры, которым мы доверяем ✅
```
**Когда:** только после подтверждения "да, всё получилось"
**Частота:** 1 раз (не показывать повторно)

### Сценарий R2: Раздел "Надёжные МФО" (уже есть — улучшить)
Добавить квиз + UTM-кнопки к существующему разделу:
```
[до 5 000] [5–15 000] [15–50 000] [более 50 000]
```
→ фильтрация по сумме → 2-3 оффера с кнопками.

### Сценарий R3: Новая кнопка "Нужен займ? 💰"
Добавить в главное меню. Ведёт в квиз → офферы.
Только для пользователей, которые НЕ пришли с жалобой на звонки.

### Сценарий R4: В еженедельной рассылке (не чаще раз в 2 недели)
```
📋 Партнёрский блок
Проверенные МФО этой недели:
1. [МФО 1] — 0% первый займ ➜
2. [МФО 2] — одобрение за 5 мин ➜
```
Показывать только тем, кто НЕ заблокировал и открывал бота последние 30 дней.

### Сценарий R5: Нативный контент "Как отличить честное МФО"
Статья → в конце: "5 МФО, которые прошли нашу проверку: [список с UTM]"

### Кому НЕ показывать рекламу МФО
- Сегмент 3 ("Спам-жертва") — они избегают МФО, реклама вызовет блокировку
- Сразу после прихода, до решения проблемы
- Более 1 раза за сессию

## Приоритеты реализации

| Приоритет | Что | Эффект | Сложность |
|-----------|-----|--------|-----------|
| 🔴 Высокий | Follow-up через 5 дней | Снизит блокировки на ~30% | Средняя |
| 🔴 Высокий | Оффер МФО после отписки (R1) | Первая монетизация | Низкая |
| 🔴 Высокий | Онбординг-вопрос при /start | Правильная маршрутизация | Низкая |
| 🟡 Средний | Квиз подбора МФО + UTM-ссылки | Основной канал монетизации | Средняя |
| 🟡 Средний | Генератор письма на отписку | Уникальная ценность | Средняя |
| 🟡 Средний | Жалоба в ЦБ / Роспотребнадзор | Уникальная фича vs конкурентов | Средняя |
| 🟡 Средний | Полезная еженедельная рассылка | Удержание, меньше блокировок | Средняя |
| 🟢 Низкий | Калькулятор экономии | Позитивная эмоция, вирусность | Низкая |
| 🟢 Низкий | История сервисов пользователя | UX для повторных визитов | Средняя |
| 🟢 Низкий | Инструкции по конкретным банкам | Полезность | Низкая |
| 🟢 Низкий | Подписка на месячные напоминания | Долгосрочное удержание | Средняя |
